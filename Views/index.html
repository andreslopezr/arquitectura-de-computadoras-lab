Arquitectura de Computadoras

A continuación podras conocer un sistema digital con tecnología microelectrónica capaz de procesar datos a partir de un grupo de instrucciones denominado programa mejor conocida como computadora. La estructura básica de esta incluye microprocesador (CPU), memoria y dispositivos de entrada/salida (E/S), junto a los buses que permiten la comunicación entre ellos.
